# Windows Screen Analyzer

A high-performance screen content recognition tool for Windows using native UI Automation APIs and local OCR.

## Features

- **Native Windows UI detection** (no OCR needed for standard controls)
- **Local OCR** for image-based text (EasyOCR, Tesseract)
- **Window-specific analysis**
- **Click automation**
- **100% offline** - no internet or API keys required

## Installation

1. Install Python 3.8+
2. Install Tesseract OCR:  
   https://github.com/UB-Mannheim/tesseract/wiki
3. Install dependencies:

```bash
pip install -r requirements.txt
```
